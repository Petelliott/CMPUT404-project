## Getting Started:
### Setup:
```
virtualenv venv --python=python3
source venv/bin/activate
pip install -r requirements.txt
```

***Note: If you are using macOS, you might need to install libmagic.***
```
brew install libmagic
```

### Run:
```
python manage.py runserver
```
